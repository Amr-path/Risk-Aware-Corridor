# Benchmark Maps

Download benchmark maps from the [Moving AI Lab](https://www.movingai.com/benchmarks/) and place them in this directory.

## Required maps for Experiment 10

### Game maps (Dragon Age: Origins)
- `dao/lak303d.map` + `dao/lak303d.map.scen`
- `dao/den501d.map` + `dao/den501d.map.scen`
- `dao/brc202d.map` + `dao/brc202d.map.scen`
- `dao/ost003d.map` + `dao/ost003d.map.scen`

### Random maps
- `random/random512-20-0.map` + `random/random512-20-0.map.scen`
- `random/random512-30-0.map` + `random/random512-30-0.map.scen`
- `random/random512-10-0.map` + `random/random512-10-0.map.scen`
- `random/random512-40-0.map` + `random/random512-40-0.map.scen`

### Maze maps (optional, for Experiment 10c)
If Moving AI Lab maze maps are unavailable, the experiment will generate procedural mazes automatically.
- `maze/maze512-1-0.map` + `maze/maze512-1-0.map.scen`
- `maze/maze512-2-0.map` + `maze/maze512-2-0.map.scen`
- `maze/maze512-4-0.map` + `maze/maze512-4-0.map.scen`
- `maze/maze512-8-0.map` + `maze/maze512-8-0.map.scen`
